# DCA Strategy Optimization — Nikkei 225

Бэктест и сравнение стратегий долларового усреднения (DCA) на индексе Nikkei 225 с учётом курса JPY/USD

## Сравниваемые тратегии

**1. Простой DCA** - 100 долларов каждый понедельник

**2. DCA с удвоением на просадке** - 200 долларов каждую неделю если портфель ниже вложенного, иначе 100 долларов

**3. Простой Value Averaging** - вкладываем столько, чтобы портфель рос на 100 долларов в неделю. Если портфель вырос — покупаем меньше, если упал — больше, в диапазоне: $10–$500

## Метрики

- ROI и CAGR
- Коэффициент Шарпа 
- % времени в просадке

Итоговое сравнение - тепловая карта с результатами по каждой метрике
